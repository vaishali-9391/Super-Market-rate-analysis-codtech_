Project Name: Supermarket Sales Analysis

Introduction : Data analysis helps businesses understand customer behavior, sales performance, and market trends. In this project, supermarket sales data is analyzed to identify sales patterns, customer preferences, and branch performance.

Objective : The main objectives of this project are:

Analyze overall supermarket sales.
Identify the best-selling product categories.
Compare sales performance among branches.
Study customer purchasing behavior.
Analyze payment methods used by customers.
Visualize sales trends using charts and graphs.
Dataset Description : The dataset contains supermarket sales transactions with the following attributes:
Invoice ID
Branch
City
Customer Type
Gender
Product Line
Unit Price
Quantity
Tax
Total Sales
Date
Time
Payment Method
Gross Income
Rating
Tools and Technologies Used :
Python
Pandas Matplotlib Seaborn
Jupyter Notebook GitHub

Data Preprocessing : The following preprocessing steps were performed:

Loaded the dataset.

Checked for missing values.

Removed duplicate records.

Converted date columns into datetime format.

Verified data types.

Prepared data for visualization and analysis.

Exploratory Data Analysis :

6.1 Total Sales Analysis :

The total sales generated by the supermarket were calculated to understand overall business performance. Result:

Total sales were obtained by summing the "Total" column.
6.2 Branch-wise Sales Analysis :

Sales performance of each branch was compared.

Observation:

Some branches generated higher revenue than others.
Branch performance helps management identify high-performing locations.
6.3 Product Line Analysis :

Sales were analyzed across different product categories.

Observation:

Certain product lines contributed more revenue.
Best-selling products can be prioritized for inventory management.
6.4 Payment Method Analysis :

Customer payment preferences were studied.

Observation:

Customers used Cash, Credit Card, and E-wallet methods.
The most preferred payment method was identified.
6.5 Customer Type Analysis :

Comparison between Member and Normal customers.

Observation:

Members generally contributed significant revenue.
Loyalty programs can improve customer retention.
6.6 Gender-wise Sales Analysis:

Sales generated by male and female customers were analyzed.

Observation:

Both genders contributed substantially to overall sales.
Helps understand customer demographics.
Data Visualization
The following visualizations were created:

Branch Sales Bar Chart
Product Line Sales Chart
Payment Method Pie Chart
Monthly Sales Trend Chart
Correlation Heatmap
These visualizations provide better understanding of sales performance and customer behavior.

Results
The analysis revealed:

High-performing branches generated greater revenue.
Certain product categories dominated sales.
Customer membership positively impacted sales.
Digital payment methods were widely used.
Sales trends varied across different periods.
Conclusion
The Supermarket Sales Analysis project successfully analyzed sales transactions and customer behavior. Through data cleaning, visualization, and statistical analysis, valuable business insights were obtained. The findings can help management improve decision-making, optimize inventory, enhance customer satisfaction, and increase overall profitability.
